## 核心知识点
- `核心必会`：tool calling 的输入输出约束。
- `核心必会`：ReAct / Plan-and-Execute / Reflection 等工作流。
- `工程重点`：提示注入、越权、错误执行链等系统风险。

## 易错点 / 边界条件
- 容易把 agent 看成“更聪明的 prompt”，忽略权限和状态管理。
- 容易让模型在高风险场景中自由决策。
