# LLM 学习模块大纲

> 共8大模块，预计16-24周完成（可根据进度调整）

---

## 模块一：Transformer 深度剖析（2-3周）
**核心问题**：Transformer为什么能成为现代LLM的基石？

| 单元 | 内容 | 诊断重点 |
|------|------|----------|
| 1.1 | Self-Attention 机制详解 | 你是否真正理解Q/K/V的数学含义？ |
| 1.2 | Multi-Head Attention 设计思想 | 知道并行计算 vs 理解表征子空间 |
| 1.3 | Positional Encoding 的演进 | 从正弦到RoPE、ALiBi |
| 1.4 | Layer Normalization & 残差连接 | 为什么Pre-norm成为主流？ |
| 1.5 | Transformer 代码实战 | 从零实现一个简洁版本 |

**模块产出**：一个从零实现的Transformer，能跑通翻译或文本生成任务

---

## 模块二：预训练技术与范式（2-3周）
**核心问题**：如何让模型"读万卷书"？

| 单元 | 内容 | 诊断重点 |
|------|------|----------|
| 2.1 | 预训练任务：MLM vs CLM vs Span Corruption | 各任务的适用场景和trade-off |
| 2.2 | 数据工程：清洗、去重、质量筛选 | 数据对模型能力的影响机制 |
| 2.3 | Tokenization：BPE、SentencePiece、BPE-dropout | 词汇表设计的深层影响 |
| 2.4 | 训练稳定性：混合精度、梯度裁剪、损失尖峰处理 | 实战经验 vs 理论 |
| 2.5 | scaling laws 与计算效率 | 数据和模型哪个该优先扩展？ |

**模块产出**：完成一次小规模预训练实验（如100M参数模型）

## 诊断性前置问题库

每个模块开始前，我会从中选取3-5个问题，根据你的回答确定讲解深度。

### 模块一诊断问题示例：
1. 你能用一句话解释Attention的数学本质是什么吗？（不是"关注重要部分"这种描述）
2. 为什么Transformer需要Multi-Head？单层大的Attention不行吗？
3. LayerNorm放在残差连接前和后有什么区别？为什么Pre-norm更稳定？
4. 你能写出Scaled Dot-Product Attention的完整公式并解释每个term的作用吗？
5. 如果让你实现一个Transformer，你会如何验证它的正确性？
